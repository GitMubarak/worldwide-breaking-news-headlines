== Installation ==

Install Worldwide Breaking News Headlines as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search Worldwide Breaking News Headlines
3. Click to install
4. Activate the plugin
5. Set the widget in Admin Widgets area.

**Install by upload plugin:**

1. Download the latest version of Worldwide Breaking News Headlines (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded worldwide-breaking-news-headlines.zip and click on "Install Now" button
6. Activate the plugin
7. Worldwide Breaking News Headlines will appear in Admin Widget area.

== Frequently Asked Questions ==

= I have installed the plugin. What next? =

Please go to Appearance -> Widgets section. Serach for 'Worldwide Breaking News Headlines' and drag it to your sidebar area. Apply settigns. 

= I have changed News source, but news feed not uppdating =

As there is a cache active for 15 minute, so you have to wait for 15 minute to see the updated news.

= How can I reduce the Cached time? =

The API we are using here has a limitation for 15 minute delay news update. So there is nothing we can do now.

= How to increase requests per day for a specific API Key? =

You can register to NewsAPI.org and get your personal API key and use it in widget API Key field. This will increase your requests per day limit.

= I have changed News source but there is no data! =

Sometimes due to main API server loading issue you may see no data. Keep refreshing until your data is properly cached.

= Any planning to add more options/newspaper? =

Yes, we are working on it. Please stay with us.
